参与或主导的核心项目。

注册制改革对A股股价崩盘风险的影响研究
基于 5229 只 A 股、2019—2023 年面板数据，采用 Pooled OLS + 聚类稳健标准误 研究注册制改革对股价崩盘风险（NCSKEW / DUVOL）的影响；通过 PCA 构建情绪指数， 发现高情绪显著降低崩盘风险，注册制改革对科创板有显著正向影响，效应集中于大盘股。
Python Stata DID 面板数据 PCA
